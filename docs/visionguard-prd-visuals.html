Section 6

用户痛点表

围绕学习、办公、创作、夜间使用和隐私顾虑梳理用户问题与产品机会。

用户场景用户痛点具体表现产品机会对应功能
学习 / 写论文连续用眼过久用户专注学习时容易忘记休息提供即时休息提醒Session Time、Rest Reminder
办公 / 编程眨眼频率下降长时间盯屏导致眨眼减少，用户自己难以察觉实时监测眨眼频率并提醒Blink Rate、Blink Reminder
设计 / 内容创作距离和坐姿不稳定用户可能长时间靠近屏幕处理细节识别距离过近并提醒调整Distance、Distance Reminder
夜间学习 / 办公环境亮度不合适过暗或过亮环境下使用屏幕，容易造成不适监测亮度并提供调整建议Brightness、Brightness Reminder
高强度屏幕使用不清楚全天累计负荷用户知道当前用了多久，但不清楚今天累计屏幕使用时长统计当日累计使用时长Daily Load、Trend、AI Report
复盘用眼习惯缺少长期趋势反馈用户无法判断自己的用眼习惯是否改善提供历史趋势和 AI 报告Trend、AI Report
摄像头产品使用隐私顾虑用户担心摄像头画面和个人数据被不当使用明确说明数据使用边界Privacy、Settings、Clear Data
